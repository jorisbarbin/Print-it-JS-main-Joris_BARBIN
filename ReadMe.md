# Print It – Projet 6 OpenClassrooms

## Objectif du projet

Ce projet a pour but de dynamiser un site web statique en JavaScript en créant un carrousel fonctionnel sur la bannière du site Print It.

L’objectif était de :
- ajouter les flèches de navigation 
- rendre les flèches interactives 
- générer les bullet points 
- changer l’image et le texte au clic 
- mettre en place un défilement infini

---

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- VS Code
- Live Server
- GitHub
- GitHub Pages
